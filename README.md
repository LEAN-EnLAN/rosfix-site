# rosFIX

Sitio multipagina para `rosFIX`, un servicio tecnico independiente de Rosario orientado a celulares, notebooks y PC.

## Stack

- React 19
- Vite
- Tailwind CSS 4
- SVG custom para hero y visuales tecnicos

## Paginas

- `index.html`
- `servicios.html`
- `proceso.html`
- `sobre-rosfix.html`
- `contacto.html`
- `diagnostico.html`

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
