# instagram_caption

Before writing, read:

- `../../references/brand-system.md`
- `../../references/business-constraints.md`

## Goal

Write captions that sound like ROSFIX, not like a generic social media manager.

## Format

Default structure:

1. problema
2. aclaracion o reencuadre
3. solucion o criterio
4. CTA simple

## Rules

- one central idea
- easy to scan
- value first
- minimal CTA
- no emoji spam
- no childish language
- no fake-premium claims

## Delivery

Return:

- final caption
- up to 2 optional variants only if they materially change angle or rhythm
