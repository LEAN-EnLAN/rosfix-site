# brand_guard

Before auditing, read:

- `../../references/brand-system.md`
- `../../references/business-constraints.md`

## Audit order

Check in this order:

1. business truth
2. identity
3. copy tone
4. visual system
5. commercial action

## Required output

Return:

- verdict: pass, revise, or reject
- findings ordered by severity
- exact rule breaks
- corrected direction

## Mandatory checks

- does it imply capabilities ROSFIX does not have
- does it sound generic or inflated
- does it drift into blue-tech or flyer aesthetics
- does it preserve clarity in under 2 seconds
- does it leave a credible next action
