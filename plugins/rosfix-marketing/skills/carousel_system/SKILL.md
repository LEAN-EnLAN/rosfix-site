# carousel_system

Before writing, read:

- `../../references/brand-system.md`
- `../../references/business-constraints.md`
- `../../references/tool-routing.md`

## Default structure

- Slide 1: strong hook
- Slide 2-4: clear development
- Last slide: closing line + contact

Default length:

- 5 slides unless the user asks for a different count

## Slide rules

- one job per slide
- on-slide copy must stay short
- visual concept must stay dominant
- CTA only on the last slide unless needed earlier for clarity

## Output

Return slide by slide:

- objective
- headline
- support copy
- visual direction
- optional caption bridge if useful

## Execution rule

If the user wants the carousel produced, prefer Canva after the structure is approved.
