# visual_prompt

Before writing, read:

- `../../references/brand-system.md`
- `../../references/business-constraints.md`
- `../../references/asset-map.md`

## Required ingredients

Always include:

- official ROSFIX logo
- Swiss grid
- terracotta identity
- cinematic studio macro realism
- realistic materials
- controlled negative space
- subtle grain

Always exclude:

- blue tones as primary identity
- generic tech startup styling
- random futuristic overlays
- cheap gradients
- stock design
- obvious AI cleanliness

## Output

Return:

- prompt final
- negative prompt or exclusions
- composition notes
- logo placement notes
- postproduction notes

## Bias

Prefer believable studio realism over spectacle.
