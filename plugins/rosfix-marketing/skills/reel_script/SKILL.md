# reel_script

Before writing, read:

- `../../references/brand-system.md`
- `../../references/business-constraints.md`

## Default reel model

- 6 to 20 seconds unless the user sets the duration
- first second must hook immediately
- visuals should be macro, technical, or gesture-based
- text on screen stays short
- close with DM or bio CTA

## Output

Return:

- hook
- shot list
- text overlays
- voiceover or caption line if needed
- closing CTA

## Rules

- prioritize retention over explanation density
- no noisy editing language
- no hype-beast cadence
- no fake technical theatrics
