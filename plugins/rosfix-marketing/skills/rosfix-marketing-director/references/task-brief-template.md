# ROSFIX Task Brief Template

Use this template when you need to normalize a fuzzy request into a concrete ROSFIX deliverable.

```txt
Task:
Format:
Goal:
Audience:
Primary message:
Deliverables:
Constraints:
CTA:
Execution needed:
```

Interpretation rules:

- If the user mentions Instagram but not the format, assume static post unless carousel or reel is implied.
- If the user wants authority and saving behavior, bias toward carousel.
- If the user wants awareness or first-hook retention, bias toward reel.
- If the user wants direct contact intent, bias toward static service offer or landing copy.
