# ROSFIX Brand System

Use this file as the single source of truth for brand behavior.

## Core job

Turn ROSFIX into a recognizable, premium, technically credible brand that generates contact intent.

The agent does not optimize for "pretty content".
It optimizes for:

- recognition
- coherence
- perceived control
- commercial clarity
- contact intent

## Brand truth

- Brand: ROSFIX
- City: Rosario
- Core business: service and diagnostics for devices and computers
- Main channel: Instagram
- Visual base: Swiss, editorial, contemporary, technical

## Positioning

- "Diagnostico antes que humo."
- "Precision antes que improvisacion."

## Visual DNA

ROSFIX must feel like:

- technical brand with judgement
- serious service
- editorial design
- controlled visual lab
- silent sophistication

ROSFIX must never feel like:

- neighborhood repair flyer
- generic blue tech brand
- gamer aesthetic
- improvised Canva template
- overloaded promo poster

## Palette

```css
--bg-primary: #0D0D0D;
--bg-secondary: #1A1A1A;
--bg-elevated: #232323;
--text-primary: #EDEDED;
--text-secondary: #A1A1A1;
--accent-main: #C47A5A;
--accent-soft: #D8A08A;
--accent-dark: #8A4F3A;
```

Color rules:

- black is the base
- soft white carries reading
- terracotta is identity, not decoration
- never use blue as primary brand color

## Typography

- SF Pro Display
- Inter
- Helvetica Neue

Behavior:

- oversized headlines
- slightly negative tracking
- short body copy
- editorial microcopy

## Layout system

- 8 px base grid
- Swiss composition
- large negative space
- strict alignment
- one dominant idea per piece

## Image treatment

Required:

- photorealistic studio feel
- macro or technical detail
- believable materials
- realistic dust or wear when useful
- high contrast
- subtle grain
- terracotta duotone when it strengthens identity

Forbidden:

- cheap glow
- neon gamer look
- fake lens flare
- pointless gradients
- obvious stock visuals
- plastic AI surfaces

## Logo rule

Always use the official ROSFIX logo or approved mark.
Never replace the logo with improvised text.

Allowed:

- main logo
- compact version
- mark "x" only as support, never as the main substitute

## Copy tone

- direct
- sober
- technically clear
- calm authority
- commercial without shouting

Most pieces should follow:

1. Hook
2. Problem reframe
3. Brief explanation
4. Minimal CTA

Valid CTAs:

- Escribinos.
- Consultanos por DM.
- Lo revisamos.
- Diagnostico claro.
- Mas info en la bio.
- Traelo y lo vemos.

## Quality tests

1. Identity: remove the name and the piece should still feel like ROSFIX.
2. Clarity: the main message should land in under 2 seconds.
3. Coherence: it must not look like a generic tech brand.
4. Realism: the image must not look like obvious AI.
5. Commercial: there must be a clear next action, even if subtle.
